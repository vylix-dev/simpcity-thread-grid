## [9.1.5] - 2026-06-26

### Changed
- Added vylix logo metadata so Tampermonkey dashboard entries use the project icon.

## [9.1.4] - 2026-06-26

### Changed
- Added the vylix logo to the Thread Grid settings modal header.

## [9.1.2] - 2026-06-26

### Fixed
- Prevented Thread Grid thumbnail sizing from leaking into search-result rows and collapsing titles into vertical text.

## [9.1.1] - 2026-06-25

### Changed
- Refined the settings nav button with an isolated custom sliders icon so site icon styles cannot override it.

### Fixed
- Removed the washed-out overlay in full-image thumbnail mode by rendering thumbnails from a single background image layer.

## [9.1.0] - 2026-06-25

### Added
- Added a Thumbnail Fit setting so cards can either crop thumbnails to fill or show the full uncropped image.

## [9.0.1] - 2026-06-25

### Changed
- Restyled the settings nav button and modal to match the vylix dark/red console design while keeping the control in the top-bar discovery area.

## [9.0.0] - 2026-06-24

### Added
- Initial repo-ready public release for `vylix-dev/simpcity-thread-grid`.
- Raw GitHub `@updateURL` and `@downloadURL` metadata for Tampermonkey updates.
- Documentation for settings, install flow, update flow, and supported domains.

### Changed
- Replaced placeholder GitHub Pages/support metadata with `vylix-dev` GitHub repository and raw install/update URLs.
- Replaced placeholder author metadata with `vylix-dev`.
